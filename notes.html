  41. Scope Rules (Variable Scopes) in Go
    1. 3 Levels of Scope
    2. Local: Declaration within functions, can be used only within that function
    3. Declartion within block e.g. if-else, Can be used only within that block
    4. Package: Declaration outside all functions, Can be used everywhere in the same package
    5. Global: Declaration outside all functions and uppercase first letter, Can be used everywhere across all packages
    6. Variable scope = Scope in the region of a program, where a defined variable can be accessed
